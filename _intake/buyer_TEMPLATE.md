# Buyer intake — canonical field set

> **This is a field-set reference, not an offline form.** Intake completion happens in chat. Use this file to know what 01 will ask about; copy fields into the chat if you've already gathered them; or just paste the lead's raw info and 01 will walk you through any gaps conversationally.

## How to use

- **Got everything?** Copy the block below, fill in what you have, paste into chat. 01 reads structured input directly.
- **Got some of it?** Paste what you have in any format. 01 prompts for missing fields.
- **Just walked out of an open house with a name and a phone number?** Paste that. 01 starts the case file and asks the rest.

## What 01 needs at minimum to open the case file

Name + intent (buying). Everything else can be classified as agent-side gaps (likely known by the agent) or lead-side gaps (need to come from the lead — 01 can hand to 03 to draft a follow-up if the lead needs to be contacted).
